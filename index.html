Часті питання

Як перевірити швидкість інтернету онлайн?
Натисни на тест "Швидкість інтернету" та запусти перевірку. Сервіс виміряє завантаження (download), вивантаження (upload) та ping до найближчих серверів у Мбіт/с.
Чи безпечні ці тести для ПК?
Так, усі тести виконуються в браузері без встановлення програм. CPU та RAM тести короткочасно навантажують систему, але не можуть пошкодити обладнання.
Як перевірити мертві пікселі монітора?
Використай тест монітора — він показуватиме однотонні кольори на весь екран. Натисни кнопку повного екрану та уважно оглянь зображення на наявність темних чи яскравих точок.
Що означає поганий ping?
Ping до 20 мс — відмінно, 20-50 мс — добре, 50-100 мс — прийнятно для гейму, 100+ мс — можуть бути затримки в іграх та відеодзвінках.
Як перевірити клавіатуру на ghosting?
Відкрий тест клавіатури та натискай кілька клавіш одночасно. Якщо деякі натискання не реєструються — це ghosting. Ігрові клавіатури з NKRO підтримують необмежену кількість одночасних натискань.
Скільки FPS повинен показувати браузер?
Нормальний результат — 60 FPS (або більше для моніторів 120/144 Гц). Якщо FPS нижче 30 при простому WebGL тесті — відеокарта або браузер потребують уваги.
Чи зберігаються мої результати?
Ні, ми не зберігаємо жодних результатів на сервері. Всі тести виконуються локально у твоєму браузері. Ми не збираємо персональні дані.
Що таке DPI мишки і як його перевірити?
DPI (dots per inch) — чутливість мишки. В тесті мишки на PCTest ти можеш рухати мишкою та порівнювати реальний рух зі значенням DPI у налаштуваннях для виявлення невідповідностей.

Про PCTest.pp.ua

PCTest.pp.ua — це безкоштовний онлайн сервіс для комплексного тестування ПК та ноутбуків. Ми зібрали 11 найпотрібніших інструментів діагностики в одному місці: від тесту швидкості інтернету до перевірки кожної клавіші на клавіатурі.

Всі тести працюють прямо в браузері без встановлення додаткового ПЗ. Ідеально для швидкої діагностики перед покупкою або продажем комп'ютера, або коли щось "гальмує".

Сервіс підтримує всі сучасні браузери: Chrome, Firefox, Edge, Safari. Працює на Windows, macOS та Linux.

PWM (Pulse Width Modulation) — метод регулювання яскравості підсвічування через швидке мерехтіння. Якщо монітор використовує PWM, це може викликати втому очей та головний біль.
